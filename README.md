📘 Prontuário UFS – Sistema de Gestão de Pacientes e Anamneses

Sistema completo de gerenciamento de pacientes, anamneses, planos alimentares e dados clínicos, desenvolvido em FastAPI, React, PostgreSQL e Docker.
Projetado para uma necessidade real de uma professora de Nutrição da univeridade Federal de Sergipe, com foco na rapidez, segurança, escalabilidade e usabilidade.

Link para acessar o site: https://prontuarioufs.pages.dev/login
